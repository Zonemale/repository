2.继承性：子元素会继承父元素所设置的样式，但不是所有的属性都会继承
3.css优先级：*权重(0000) < div权重(0001) < class权重(0010) < id权重(0100) < !important权重无限大(示例：color:red!mportant;)
    1）权重相同则就近原则
    2）权重会叠加，div p: 0001+0001=0002 > p: 0001
    3)继承的权重为0

六、定位(position)
1.边偏移：top,bottom,left,right
2.定位模式(定位的分类)
    1）static:自动定位，静态定位（默认所有元素都是静态定位），
    作用：取消定位。
    2）relative:相对定位，相对其原文档流的位置进行定位（自恋型）
    以自己的左上角位置为参考进行移动。原来位置的空间仍被保留
    3）absolute:绝对定位，相对于其上一个已经定位的父元素进行定位（拼爹型）
    不占原有位置，以整个窗口的左上角为参考
    如果父级没有定位，则以可视窗口为参考定位，如果父元素有定位，则会依据最近的已经定位的父元素进行定位
    4）fixed：固定定位，相对于浏览器窗口进行定位，只认浏览器(认死理型)
    注意：子绝父相（子元素为绝对定位，父元素为相对定位）
3.定位盒子居中对齐
    1）加了定位和浮动的盒子margin: 0 auto;就会失效
    2)首先left: 50%,父盒子一半大小，然后走自己外边距负的一半值就可以了,margin-left.
4.定位模式转换：跟浮动一样，元素添加了绝对定为和固定定位（position:absolute,fixed)之后，元素模式也会发生转换，都转换为行内块模式，因此比如行内元素如果添加了绝对定位或者固定定位后浮动后，可以不用转换模式，直接给高度和宽度值就可以了。
5.叠放次序（z-index)
    1)只有绝对定位，相对定位，固定定位有此属性，浮动，静态定位都无此属性
6.vertical-align: 垂直对齐
7.text-overflow:文字溢出
8.display:none inline block;隐藏元素后位置不再占据
9.visibility:visible hidden隐藏元素后仍占据位置
10.overflow溢出隐藏：
    1）值：visible auto hidden scroll

十一、css3过渡动画和帧动画
1.过渡动画：是从一个状态主键的过渡到另外一个状态（多组属性用逗号分割）
    1)transition:简写属性，用于在一个属性中设置四个过渡属性
    2）transition-property:规定应用过渡的css属性的名称
    3）transition-duratioin:定义过渡效果花费的时间，默认是0
    4）transition-timing-function:规定过渡效果的时间曲线，默认是"ease"(linear,ease,ease-in,ease-out)
    5)transition-dely:规定过渡效果何时开始，默认是0
    示例：
        div{
            width: 100px;
            height: 100px;
            background-color: pink;
            transition: width 2s ease-in 0s;
        }
        div:hover{
            width: 400px;
        }
    注意：过渡动画写到本身上，谁做过度动画写到谁身上，但是也可以写到hover里，效果不好
    一般写法（transition:all 1s;)
2.帧动画：通过一帧一帧的画面按照固定顺序和速度播放
十二、2d变形（transform)
1.translate(x,y):移动
    translate如果x -50% 跟父亲没关系，是走自己盒子宽度的一半
2.scale(x,y):缩放
3.rotate(deg):旋转,正值为顺时针，负值为逆时针，
    transform-origin可以调整元素转换变形的原点,示例：transform-origin: bottom left;
4.skew(deg,deg):倾斜
十三、css3动画原理animation
1.animation:动画名称，动画时间，运动曲线，何时开始，播放次数(infinite规定播放无限次数)，是否反方向（否：normal，是：alternate);
    1)示例:
        a.div{
            width: 100px;
            height: 100px;
            background-color: pink;
            position: absolute;
            left: 0;
            /*调用动画*/
            animation: move 3s ease 0s 2;
        }
        /*声明动画 关键帧*/
        @keyframes move{
            from{
                left: 0;
            }
            to{
                left: 1000px;
            }
        }
        b.img{
            width: 400px;
            height: auto;
            animation: move 0.5s infinite;
        }
        @keyframes move{
            0%{
                transform: scale(1);
            }
            50%{
                transform:scale(1.1);
            }
            100%{
                transform: scale(1);
            }
        }
