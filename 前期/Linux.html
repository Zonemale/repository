二、Linux 的版本
  1. Linux的版本分为两种：内核版本和发行版本
  2. Linux目录结构：
    1. "/"：代表根目录
  3. Linux列出文件列表
    1. ll:列出文件列表
    2. ls:查看文件，蓝颜色为文件夹，白颜色为文件
    3. ls -a:查看所有文件，包括隐藏目录，隐藏文件或文件夹是以“.”开头的文件
    4. ls -l:显示文件详细信息
  4. 切换目录命令：cd
    1. pwd:查看当前目录
    2. cd ..  :返回上一级目录
    3. cd ~ ：返回root目录
    4. cd /*:退回到某一个目录
    5. cd /:切换到用户主目录
    6. cd -:返回上一次所在的目录
    7. Linux中绝对路径：cd /*
    8. 相对路径：cd ../
  5. 创建目录和移除目录：mkdir------rmkdir
    1. mkdir(make directory)命令可以用来创建子目录
      1. mkdir app      在当前目录先创建app目录
      2. mkdir -p app2/test     级联创建app2以及test目录
      3. mkdir --help       提示参数信息
    2. rmdir(remove directory)命令可用来删除“空”的子目录：rmdir app-----删除app目录
  6. 浏览文件
    1. cat:用于显示文件内容，格式：cat[参数]<文件名>
      1. cat yum.conf
      2. cat a.txt > b.txt----------将a中的内容复制到b.txt中
      3. cat a.txt >> b.txt---------将a中的内容追加到b.txt中
    2. more：一般用于要显示的内容会超过一个画面长度的情况，
      1. 按空格键显示下一个画面
      2. 回车显示下一行内容
      3. 按q键退出查看
        1. 空格显示下一页数据，回车显示下一行的数据
    3. less:用法和more类似，不同的是less可以通过PgUp、PgDn键来控制
      1. less yum.conf
        1. PgUp和PgDn进行上下翻页
    4. tail：用于显示文件后几行的内容
      1. tail -10 /etc/passwd---------------查看后10行数据
      2. tail -f catalina.log-----------------动态查看日志(*****)
      3. ctrl + c:结束查看
  7. 文件操作
    1. rm：删除文件
      1. 用法：rm [选项] 文件-------------rm a.txt:删除a.txt文件
      2. 删除需要用户确认，y/n
        1. rm删除不询问
          1. rm -f a.txt---------不询问，直接删除rm删除目录
          2. rm -f a-------------递归删除不询问递归删除（慎用）
          3. rm -rf a------------不询问递归删除
          4. rm -rf *------------删除所有文件
          5. rm -rf /*-----------自杀
    2. cp(copy)命令可以将文件从一处复制到另一处，一般在使用cp命令时将一个文件复制成另一个文件或复制到某目录时，需要指定源文件名与目标文件名或目录
      1. cp a.txt btxt-------将a.txt 复制为b.txt文件
      2. cp a.txt ../---------a.txt文件复制到上一层目录中
      3. cp -r ./aa ./b---------将aa文件夹拷贝到b 文件夹中
    3. mv移动或者重命名
      1. mv a.txt ../--------将a.txt文件移动到上一层目录中
      2. mv a.txt b.txt-----将a.txt文件重命名为b.txt
    4. tar命令（打包或解压）
      1. tar命令位于/bin目录下，它能够将用户所指定的文件或目录打包成一个文件，但不做压缩。一般Linux上常用的压缩方式是选用tar将许多文件打包成一个文件，再以gzip压缩命令压缩成
      2. xxx.tar.gz(或称xxx.tgz)的文件，常用参数
        1. -c:创建一个新tar文件
        2. -v:显示运行过程的信息
        3. -f:指定文件名
        4. -z:调用gzip压缩命令进行压缩
        5. -t:查看压缩文件的内容
        6. -x:解开tar文件
        7. 打包：tar -cvf xxx.tar ./*
        8. 打包并且压缩：tar -zcvf xxx.tar.gz ./*
        9. 解压：
            1. tar -xvf xxx.tar
            2. tar -zxvf xxx.tar.gz -C /usr/aaa
        10. 注：文件类型
          1. xxxx.tar.gz:压缩并打包后的文件
          2. xxxx.tar:打包后的文件
      3. find命令：find指令用于查找符合条件的文件
        1. find / -name "ins*":查找文件名称使以ins开头的文件
        2. find / -name "ins*" -ls
        3. find / -user itcast -ls:查找用户itcast的文件
        4. find / user itcast -type d -ls:查找用户itcast的目录
        5. find / -perm -777 -type d-ls：查找权限是777的文件
      4. grep文件：查找文件里符合条件的字符串：用法：grep [选项] ... PATTERN [FILE] ...
        1. grep lang anaconda-ks.cfg-----在文件中查找lang
        2. grep lang anaconda-ks.cfg --color-----高亮显示
        3. grep w anaconda-ks.cfg --color -A1---------显示本行加上前一行
        4. grep w anaconda-ks.cfg --color - A1 -B1-------显示本行加上前后两行
  8. 其他常用命令
    1. pwd:显示当前所在目录
    2. touch:创建一个空文件
    3. clear/crtl+L:清屏
三、Vi和Vim编辑器
  1. Linux中可编辑的文件都是文本类型的
  2. 编辑器提供了三种模式
  3. Vim编辑器
    1. 三种模式
      1. 命令行：Esc键，切换到命令行模式
      2. 插入：按i、o、a键
        1. i：在当前位置前插入
        2. I：在当前行首插入
        3. a：在当前位置后插入
        4. A：在当前行尾插入
        5. o：在当前行之后插入一行
        6. O：在当前行之前插入一行
      3. 底行模式：按“:”（冒号）
      4. 其他命令
        1. 打开文件：vim file
        2. 退出：esc------->:q
        3. 修改文件：输入i进入插入模式，保存并退出：esc------>:wq
        4. 不保存退出：esc---->:q!
      5. 进入插入模式：
        1. i：在当前的光标所在处插入
        2. o：在当前光标锁在的行的下一行插入
        3. a：在光标所在的下一个字符插入
      6. 快捷键：
        1. dd-----快速删除一行
        2. yy------复制当前行
        3. nyy-----从当前行向后复制几行
        4. p-----粘贴
        5. r----替换
        6. :/-----搜索内容，高亮显示

七、Linux常见网络操作
  1. 主机名配置
    1. hostname:查看主机名
    2. hostname xxx:修改主机名，重启后无效
    3. 如果想要永久生效，可以修改/etc/sysconfig/network文件
  2. IP地址配置
ifconfig 查看(修改)ip地址(重启后无效)
ifconfig eth0 192.168.12.22 修改ip地址
如果想要永久生效
修改 /etc/sysconfig/network-scripts/ifcfg-eth0文件----------网卡文件位置
DEVICE=eth0 #网卡名称
BOOTPROTO=static #获取ip的方式(static/dhcp/bootp/none)

HWADDR=00:0C:29:B5:B2:69 #MAC地址
IPADDR=192.168.177.129 #IP地址
NETMASK=255.255.255.0 #子网掩码
NETWORK=192.168.177.0 #网络地址
BROADCAST=192.168.0.255 #广播地址
NBOOT=yes #  系统启动时是否设置此网络接口，设置为yes时，系统启动时激活此设备。
  1. 域名映射
/etc/hosts文件用于在通过主机名进行访问时做ip地址解析之用,相当于windows系统的C:\Windows\System32\drivers\etc\hosts文件的功能
  1. 网络服务管理
service network status 查看指定服务的状态
service network stop 停止指定服务
service network start 启动指定服务
service network restart 重启指定服务
service --status–all 查看系统中所有后台服务
netstat –nltp 查看系统中网络进程的端口监听情况
防火墙设置
防火墙根据配置文件/etc/sysconfig/iptables来控制本机的”出”、”入”网络访问行为。
service iptables status 查看防火墙状态
service iptables stop 关闭防火墙
service iptables start 启动防火墙
chkconfig  iptables off 禁止防火墙自启
八、Linux上软件安装
  1. Linux上的软件安装有以下几种
    1. 二进制发布包：软件已经针对具体平台编译打包发布，只要解压，修改配置即可
    2. RPM包：软件已经按照redhat的包管理工具规范RPM进行打包发布，需要获取到相应的软件RPM发布包，然后用RPM命令进行安装
    3. Yum在线安装：软件已经以RPM规范打包，但发布在了网络上的一些服务器上，可用yum在线安装服务器上的rpm软件，并且会自动解决软件安装过程中的库依赖问题
    4. 源码编译安装：软件以源码工程的形式发布，需要获取到源码工程后用相应开发工具进行编译打包部署。
  2. 上传与下载工具
    1. FileZilla：
    2. lrzsz：yum install lrzsz
      1. 上传：rz
      2. 下载：sz
  3. sftp
    1. alt + p:组合键打开sftp窗口
    2. 使用put命令上传----------put H:\uploads\03.jdk（会上传到操作目录，也就是root目录下）
    3. 使用get命令下载----------get 03.jdk（会下载到本地文档目录下）
  4. rpm -qa | -gtep xxx-----------查询已安装的所有软件包
  5. rpm -e -nodeps xxxrpm -----------卸载软件包
  6. rpm指的是软件包管理器
九、软件安装与卸载
  1. JDK安装与卸载
    1. 卸载默认安装的jdk：
      1. rpm -e --nodeps java-1.6.0-openjdk-1.6.0.35-1.13.7.1.el6_6.i686
    2. 直接解压自己的jdk
      1. tar -xvf jdk.tar.gz -C 目标路径
    3. 配置jdk环境变量
      1. vi /etc/profile
      2. 修改：JAVA_HOME=/usr/local/jdk/jdk1.7.0_71
      3. source /etc/profile -------使更改的配置立即生效
  2. 安装mysql
    1. 上传mysql
    2. 卸载Linux自带的mysql
      1. rpm -qa | grep mysql
      2. rpm -e --nodeps mysql-libs-5.1.73-5.e16_6.i686
    3. 安装mysql
      1. 安装MySQL服务端：rpm -ivh MySQL-server-5.6.22-1.e16.i686.rpm
      2. 安装MySQL客户端：rpm -ivh MySQL-client-5.6.22-1.e16.i686.rpm
      3. 修改密码：set password=password('root')
      4. 开启mysql服务：service mysql start
    4. mysql远程访问设置
      1. grant all privileges on *.* to 'root' @'%' identified by 'root';
      2. flush privileges;
    5. 关闭防火墙：service iptables stop
  3. 安装tomcat
    1. 上传解压
  4. 安装redis
    1. 下载源码进行编译：yum install gcc-c++
    2. 下载redis：wget  http://download.redis.io/releases/redis-3.0.4.tar.gz
    3. 解压：tar -xzvf redis-3.0.4.tar.gz
    4. 编译安装：
      1. cd redis-3.0.4
      2. make
    5. 执行安装命令
      1. make PREFIX=/usr/local/redis.install
    6. 配置redis：
      1. cd redis-3.0.4
      2. cp redis.conf /usr/local/redis/bin
    7. 启动redis
      1. cd redis/bin
      2. ./redis-server redis.conf
      3. ./redis-cli
